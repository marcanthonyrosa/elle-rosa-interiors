# Elle Rosa Design

Marketing website for Elle Rosa Design, a residential interior design studio based in Houston, TX.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local development

Requires Node.js and npm.

```sh
git clone <YOUR_GIT_URL>
cd elle-rosa-interiors
npm install
npm run dev
```

## Deployment

Deployed via Railway. Push to `main` to trigger a production build.
